# chen - 电子信息工程应届生简历

这是一个简约专业的个人技术简历单页网站，使用原生HTML、CSS、JS开发，支持深色模式切换、平滑滚动和卡片动画。

## 功能特点

- 响应式设计：适配手机和电脑
- 深色模式切换：一键切换明暗主题
- 平滑滚动：页面内导航流畅
- 卡片动画：项目卡片进入动画效果
- 简约专业风格：现代干净的设计

## 部署到 GitHub Pages

1. 将此仓库推送到 GitHub。
2. 在仓库设置中启用 Pages，选择 `main` 分支的根目录。
## 文件说明

- `index.html`：主页结构
- `styles.css`：样式表（包含深色模式和动画）
- `script.js`：交互脚本（主题切换和动画）

## 自定义

你可以修改 `index.html` 中的个人信息、技能和项目内容来自定义简历。
