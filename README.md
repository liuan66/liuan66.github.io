# 推广链接合集页

这是一个适合做推广引流的单页站点模板。访问者点击链接后会跳转到官方平台进行购买或注册，你可以通过推广计划获得佣金。

## 功能

- 链接卡片合集：集中展示你的推广入口
- 分类筛选：按电商、工具、学习等维度查看
- 深色模式：自动记住用户偏好
- 本地点击统计：记录站内按钮点击次数
- 合规披露文案：默认包含推广说明

## 快速修改

1. 打开 script.js。
2. 修改 links 数组中的 title、desc、category、url。
3. 将 url 换成你的真实推广链接。
4. 保存并发布到 GitHub Pages。

## 部署到 GitHub Pages

1. 将仓库推送到 GitHub。
2. 在仓库 Settings -> Pages 中选择主分支根目录发布。
3. 打开你的站点链接进行访问测试。

## 合规建议

- 保留页面中的推广披露说明。
- 不夸大收益或虚假承诺。
- 遵守目标平台的推广政策与法律法规。
